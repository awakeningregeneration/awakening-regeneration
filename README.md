# Awakening Regeneration

Awakening Regeneration is a living map that helps people discover and support regenerative businesses, projects, and initiatives already improving life in their region.

The goal is simple: make the life-supporting work already happening around us visible, accessible, and easier to participate in.

## Current MVP Features

- Interactive Mapbox map
- Local regenerative listings ("lights")
- County-aware regional context
- Sidebar exploration and search
- Submission page for new listings

## Vision

Awakening Regeneration is designed to become a living network where people can:

- Discover regenerative businesses near them
- Support projects improving ecological and community health
- Share stories of regeneration happening locally
- Participate in redirecting attention and resources toward life-supporting systems

## Technology

Built with:

- Next.js
- TypeScript
- Mapbox GL
- (Upcoming) Supabase for database persistence
